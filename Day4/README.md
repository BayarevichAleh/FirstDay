## Функция вычисления f(x) в зависимости от значения x (Function_2.py)

f(x)= x^2 при -2<=x<2;
x^2+4x+5 при x>=2;
4 при x<-2.
Используя эту функцию для n заданных чисел, вычислить f(x). Среди вычисленных значений найти наибольшее.<br>
